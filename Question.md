# 想定される質問一覧
## 面接でのテクニック
### 1. 「はい/いいえ」で答えられる質問に対しては、まず「はい」か「いいえ」で答える。ダメな例の典型が、「はい」か「いいえ」を答えずに長々と説明を始めるパターン。
### 2. 質問への回答は端的かつ簡潔に。質問の意図がよくわからない場合は、「……という意味ですか？」と質問意図を聞き返して確認する。
### 3. 研究の根本に関わる部分や前提・仮定の部分以外で、わざと説明を省略しておき、質問するように仕向ける、という高等テクニックもある。初心者にはお薦めしない。

| segmentationとは | 画像の各画素がどのカテゴリに属するかを求め，マスクをかける手法のこと |
| instace segmentationとは | インスタンスごとにセグメンテーションをする手法のこと |
| セグメンテーションの手法には他にどんなのがある，それ説明できる | FCN(Fully Convolutional Netwrok : 全層畳み込みネットワーク)・FPN(Feature Pyramid network : 特徴ピラミッドネットワーク) です．FCNは最後の畳み込み層で得られた特徴マップを入力画像と同じサイズにアップサンプリングしてピクセルごとのクラス確率を出力する．・畳み込みによって得られた特徴マップをピラミッド型に伝播させていく手法．|
| U-netがセグメンテーションしたい対象が重なっている場合セグメンテーションし辛いのはなぜ | 重なっている場合は対象が単体としての形を保っていないので，別のクラスと判断してしまうことがあるから． |
| self attentionとは |ある一つの文章・画像を考えた時の,  単語間もしくは画像パッチ間の類似度を表すスコア |
| attentionとは | 異なるデータの単語間もしくは画像パッチ間の類似度を表すスコア |
| U-netとは | 完全畳み込みネットワークの一つで、画像のセグメンテーションを行うことができる. 特徴としては,アーキテクチャがU次の構造をしており，畳み込み層によって失われるベクトルの位置情報の消失を防ぐために,<br>ダウンサンプリング前の高解像の情報をアップサンプリング後に伝えるSkip-connectionをもつ. |
| U-net++ とは | skip connection上に畳み込み層を加えることで，エンコーダーとデコーダ間の特徴マップの表現上のギャップをなくすことができる． |
| skip connectionとは | ダウンサンプリング前の高解像の情報をアップサンプリング後に伝える. |
| FCN(完全畳み込みネットワーク)とは | 一般的なCNNの全結合層を畳み込み層に置き換えたもので, CNNの出力が「物体が何であるか」を出力するのに対し, 「物体がどこにあるか」を出力できる. |
| mask R-CNNとは | Faster R-CNNのバウンディングボックス回帰・クラス分類に加えマスク推定ブランチを追加したインスタンスセグメンテーション手法のこと. |
| faseter R-CNNとの違いは | ROI Alignの出力をfaster R-CNNのバウンディングボックス回帰・物体のクラス分類に加え, 物体のマスク推定ブランチを追加しているところ. |
| ROI Poolingとは | 特徴マップに投影したregion proposalとのサブピクセルのズレを解消する手法のこと． |
| ROI Alignとは | バイリニア補間によって特徴マップから正確にRegion proposalを抽出する手法 |
| バイリニア補間とは | region proposalをそのまま特徴マップに投影し，サブピクセル座標と近傍四画素との距離を求め, 加重平均値を用いて補間する方法 |
| 全結合層とは | 入力されたデータ全てを使って計算する層 |
| Region Proposal Networkとは | 入力画像から物体が存在し得る画像領域の候補を抽出するためのネットワーク |
| 物体候補領域とは | この小論文の例だと, 病変の疑いがある領域を提案してくれるということ |
| 単語埋め込みとは | 単語や画像パッチを固有のベクトルにエンコードする処理のこと |
| 位置エンコーディングとは | ベクトルに位置情報をもたせる処理のこと |
| 肺小結節とは | 肺にできるしこりのようなもの． |
