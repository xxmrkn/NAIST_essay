# 想定される質問一覧
## ・小論に関する質問
### [単語]
| 質問 | 回答 |
| :---- | :---- |
| segmentationとは | 画像の各画素がどのカテゴリに属するかを求める手法のこと |
| instace segmentationとは | 画像の各画素がどのカテゴリに属するか, どのインスタンスに属するかを求める手法のこと |
| self attentionとは |ある一つの文章・画像を考えた時の,  単語間もしくは画像パッチ間の類似度を表すスコア |
| attentionとは | 単語間もしくは画像パッチ間の類似度を表すスコア |
| U-netとは | 完全畳み込みネットワークの一つで、画像のセグメンテーションを行うことができる. 特徴としては,畳み込み層によって失われるベクトルの位置情報の消失を防ぐために,<br>ダウンサンプリング前の高解像の情報をアップサンプリング後に伝えるSkip-connectionをもつ. |
| skip connectionとは | ダウンサンプリング前の高解像の情報をアップサンプリング後に伝える. |
| FCN(完全畳み込みネットワーク)とは | 一般的なCNNの全結合層を畳み込み層に置き換えたもので, CNNの出力が「物体が何であるか」を出力するのに対し, 「物体がどこにあるか」を出力できる. |
| mask R-CNNとは | Faster R-CNNのバウンディングボックス回帰・クラス分類に加えマスク推定ブランチを追加したインスタンスセグメンテーション手法のこと. |
| faseter R-CNNとの違いは | ROI Alignの出力をfaster R-CNNのバウンディングボックス回帰・物体のクラス分類に加え, 物体のマスク推定ブランチを追加しているところ. |
| ROI Alignとは | バイリニア補間によって特徴マップから正確にRegion proposalを抽出する手法 |
| バイリニア補間とは | 周囲四画素の輝度値を参照し, その加重平均値を用いて補間する方法 |
| Fully connected layerとは | 入力されたデータ全てを使って計算する層 |
| Region Proposal Networkとは | 入力画像から物体が存在し得る画像領域の候補を抽出するためのネットワーク |
| region proposalとは | この小論文の例だと, 病変の疑いがある領域を提案してくれるということ |
| word embeddingとは | 単語や画像パッチを固有のベクトルにエンコードする処理のこと |
| positional encodingとは | ベクトルに位置情報をもたせる処理のこと |

### [処理の詳細に関して]
| 質問 | 回答 |
| :---- | :---- |
| なぜ他の手法ではなく,Mask R-CNNにしたの？ | 人間の身体のような複雑に入り組んだ画像をセグメーテンションするのに優れた手法だと考えたからです. |
| なぜself attention | 入力ベクトル内でのattentionスコアを求めることができ, <br>画像と文章のどこに注目しているのかを可視化できるから. |
| attentionはなぜ良い結果が出るの？ | 文章や画像の特定の部分に注意を向けるよう学習していくので, より人間に近い形で画像や文章を認識することができるから. |
| region propsal networkにテキストを使わないのは何故か | RPNは物体が写っている場所とその矩形の形を検出するモデルであり, あくまで提案なのでテキストによる情報は無くてもよく, 計算を減らすため. |
| U-netではなぜ位置情報が失われるの | 畳み込み層でダウンサンプリングが行われるから |

### [その他]
| 質問 | 回答 |
| :---- | :---- |
| 研究テーマの魅力について語って | 文章や画像のどこに注意してセグメンテーションを行なっているのかを可視化できるので, 患者に対する説明が容易になる. |

| この手法のメリット・デメリット教えて | メリット：通常のmask-RCNNよりセグメンテーションを正確に行うための情報を多く保持しているため, 良い結果が予想される. <br>デメリット：テキストと画像についてself attentionを求めるので計算量が大きくなる. |
| データセットはどうする | NIHのDeepLesion(ディープリーション),(タスクに依存するが, NAISTで利用できるような医療情報データベースを使いたい) |
| それって実現できる | 医用画像ではないが, 似たような研究として文章と画像から<br>セグメンテーションを行うものがあるので, 可能だと思う. |
| 新規性を出すために工夫したところは | 入力として画像と文章を使い,  attentionを算出することで, mask R-CNNのROI alignの入力の一つとして利用したところ. |
| どうして入力にテキストを加えることで精度が向上すると思ったの | 通常のmask R-CNNのようにCNNによって得られた特徴量のみを用いた場合と比べ, テキストによる情報を付加することでより詳しく画像を解釈できるようになると考えたから. |
| 引用してる論文以外にサーベイした論文はあるか | あります, Vision Transformerが提案された論文と, 佐藤先生,大竹先生も研究されてた骨盤放射線画像のセグメンテーションに関する論文. |
| この研究は何の役に立つか | 医者の診断ミスを低減させ,負担を減らせる. 加えて文章と画像のどこに注意したかを可視化できるので, 患者に対する説明可能性が上がる. |
| なぜ生体医用画像を使った研究に興味を持ったのか | 高校の時に患った難病と闘病していく中で, 主治医や他の医者の大変そうな姿を見てきて, 少しでも負担を減らせないかと考えるようになったから. また, 診察で自分のCT画像やレントゲン画像を見せられ説明されることが多くあったが, 素人には理解が及ばないことが頻繁にあり, よりわかりやすくできないかと考えるようになったから. |
